# Healthcare Analytics Dashboard

## Project Overview
Developed an interactive healthcare analytics dashboard using Power BI by integrating patient, ICU, financial, and operational datasets to provide real-time insights for hospital decision-making.

## Key Features
- Patient Flow Analysis
- ICU Monitoring
- Bed Occupancy Tracking
- Readmission Rate Analysis
- Average Length of Stay (LOS)
- Cost Per Patient Analysis
- Predictive Analytics for Resource Allocation

## Business Impact
- Improved operational visibility across hospital departments
- Supported proactive resource allocation and capacity planning
- Enhanced decision-making using KPI-driven insights
- Reduced operational inefficiencies through predictive analytics

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization
- Predictive Analytics
- KPI Reporting

## Key Outcomes
- Analyzed key healthcare KPIs including bed occupancy, patient vital signs, readmission rates, average LOS, and cost per patient to improve operational efficiency and patient care outcomes.
- Implemented AI-driven predictive analytics to forecast bed occupancy trends and identify at-risk patients for proactive healthcare management.
